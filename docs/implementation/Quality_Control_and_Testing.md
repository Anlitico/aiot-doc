# 质量控制与量产测试标准 (Quality Control & Testing)

本指南定义 500 台小批量产期间的产线自检标准，确保“每一台公仔都是完美的”。

---

## 1. 产线测试工位 (Test Stations)

### 工位 A：PCBA 功能电测 (主板打样后)
- **供电测试**：上电电压是否稳定在 3.3V ± 0.1V。
- **充电测试**：插入 USB 后，充电电流是否符合设计值 (如 500mA)。
- **基本连通性**：蓝牙/Wi-Fi 搜号是否正常。

### 工位 B：声学密封性测试 (组装后)
- **扫频测试 (Audio Sweep)**：播放 20Hz - 20kHz 扫频信号，监听是否有结构共振（滋滋声）。
- **回声消除验证 (AEC Check)**：在最大音量播放时，喊唤醒词，考核“打断成功率”。
- **底噪检测**：在静室环境中检测录音信噪比，排除 LED 干扰。

---

## 2. 关键质量标准 (Pass/Fail)

| 测试项目 | 判定标准 | 备注 |
| :--- | :--- | :--- |
| **外观缺陷** | 硅胶分模线溢料 < 0.2mm；无黑点/异色。 | 视觉检查 |
| **按键手感** | 触发力 3N ± 1N；回弹干脆，无阻塞。 | 触觉检查 |
| **Wi-Fi 信号** | 离路由器 5 米，RSSI > -65dBm。 | 射频检查 |
| **续航一致性** | 抽检 5%，满电播放时间偏差 < 10%。 | 长期可靠性 |

---

## 3. 抽检与老化测试 (Burn-in)

- **老化测试**：从 500 台中随机抽取 20 台，进行 **24 小时连续语音播报测试**，监测是否死机或发热过大。
- **跌落抽检**：随机抽取 5 台，进行 1 米高度 6 面跌落，检查结构是否松动。

---

## 4. 包装完整性 (OQC)
- [ ] 检查礼盒内是否包含：公仔、充电线、感谢卡、说明书。
- [ ] 礼盒外条形码/SN 是否与内部机器一致。
