# 供应链寻源与组装策略 (Supply Chain & Sourcing Guide)

针对本项目 500 台量级的试产，建议采用“模块化分工 + 核心环节直控”的策略。

---

## 1. 供应链角色分布图 (Sourcing Map)

| 环节 | 合作对象类型 | 交付核心 | 获取途径 |
| :--- | :--- | :--- | :--- |
| **电路主板 (PCBA)** | 专业打样/小批量加工厂 | 焊好的电路板 (母板+副板) | 嘉立创、PCBWay、捷配 |
| **硅胶外壳 (Shell)** | 液体/固体硅胶模具厂 | 符合公差的外壳蒙皮 | 1688 搜索“硅胶开模定制” |
| **塑料支架 (Chassis)** | 3D 打印或复模厂 | 内部固定骨架 | 未来工场 (3D打印快速验证) |
| **标准件 (Parts)** | 代理商/贸易商 | 电池、喇叭模组、FPC排线 | 淘宝 (散样)、1688 (大货) |
| **总装 (Assembly)** | 组装外协或硅胶厂代工 | 包装好的成品玩具 | 实地考察工厂生产线 |

---

## 2. 为什么不选择“一站式主板集成商”？

1. **研发费陷阱**：主板方案商往往要求 3-5 万的 NRE (研发费)，而我们的 ESP32-S3 方案逻辑简单，不值得支付这笔费用。
2. **供应链黑盒**：集成商为了降本，可能在电池、音频芯片上克扣质量，导致你第一批货口碑崩坏。
3. **响应速度**：小单在大型集成商那里得不到排产，自己通过嘉立创下单，5 天就能出货。

---

## 3. 关键：谁来负责“总装” (The Integrator)？

对于 500 台，你有三个实操建议：

### 建议 A：寻找“具有后端服务能力”的硅胶厂 (最优解)
很多硅胶厂（如东莞一带）承接礼品、音箱定制。
- **流程**：你寄给他们 PCBA 和电池 -> 他们生产硅胶壳 -> 并在现场完成内胆嵌入 -> 封口 -> 装入礼盒。
- **优点**：物流成本低，责任明确（如果装配坏了是他们的事）。

### 建议 B：找一家专业的“后焊/组装”小厂
专门承接 100-1000 台的智能硬件组装。
- **考察点**：是否有简单的静电防护（静电手环、静电衣）、是否有音频测试位。

---

## 4. 实施阶段：第一张询价单怎么写？

当你联系供应商时，请准备好以下资料：
1. **BOM 表**：零件详细清单（你手里的 `Pin_Mapping_Guide` 的延伸）。
2. **3D 图纸**：`.stp` 格式（由结构工程师完成）。
3. **包装要求**：彩盒、说明书、内衬材质。
4. **测试要求**：开机灯是否亮、Wi-Fi 能否连上。

---

## 5. 500 台规模的利润与风险控制
- **溢价控制**：由于量小，零件采购成本会比大厂高 20% 左右，需预留足够的利润空间。
- **品质抽检**：无论找谁组装，头 50 台必须你亲自在现场盯着看。
