# 实施与落地路线图 (Strategy & Implementation Roadmap)

本指南解决如何将纸面设计转化为实物产品，从零经验起步，逐步打通深圳供应链。

---

## 1. 核心执行策略：自己做 vs. 找外包

针对本项目 (500台试产)，建议采用以下组合：
- **硬件 (EE)**：
    - **方案 A (推荐)**：**自己做全集成 PCBA**。利用“嘉立创 EDA”克隆原理图。成本最低，集成度最高。
    - **方案 B (用户提议)**：**采购成品开发板 (如启明云端) + 自研底板**。适合极其担心硬件失败的初学者。单机成本会增加约 30-50 元，且需解决电池充电和结构堆叠问题（详见下文深度分析）。
- **结构 (MD)**：**找“小设计工作室”或“自由职业工程师”**。在闲鱼或硬派等平台找，花费 3k-5k 元将我们的文档转化为 3D .stp 文件。
- **外观 (ID)**：**在已定稿的文档基础上微调**。
- **固件 (FW)**：**自己做（AI 辅助）**。基于 ESP-ADF 框架集成你已有的软件。

---

## 2. 三步走落地时间轴

### 第一阶段：功能验证机 (EVT - 工程样机)
- **动作**：买一套 ESP32-S3-Korvo-1 开发板，连接你现有的软件。
- **配套**：淘宝买一个 3D 打印的粗糙外壳，验证背部出音的音质。
- **周期**：2 周。
- **预算**：¥500 元以内。

### 第二阶段：工程打样 (DVT - 设计验证)
- **动作**：找结构师出 3D 图 -> 找模具厂开“复模” (真空复模，无需开钢模)。
- **硬件**：下单制作我们的“母女板”，由嘉立创完成 PCBA。
- **周期**：4 周。
- **预算**：¥5,000 - ¥10,000 元。

### 第三阶段：500台小批量 (Pilot Run)
- **动作**：开铝模（硬质硅胶常用） -> 手工装配。
- **工厂**：联系深圳宝安/沙井的小型硅胶加工商。
- **预算**：¥30,000 - ¥50,000 元。

## 3. 深度决策：量产直接用开发板可行吗？(Dev Board for Mass Production)

你提到的 **“直接采购启明云端 WT9932S3-NANO (¥45.9) 进行量产”** 是一种典型的 **“核心板 + 底板” (SOM + Carrier Board)** 策略。对于 500 台的规模，这是一个**可行但有代价**的折中方案。

### ✅ 方案优势
1.  **回避射频风险**：开发板已通过 FCC/CE 认证，天线射频已调优，不用担心信号差。
2.  **降低焊接难度**：核心的 BGA/密脚芯片已贴好，你只需要制作一块简单的“底板”来连接外设。
3.  **量产烧录方便**：开发板自带 USB 串口，生产线工人直接插 USB 就能刷固件，无需制作烧录夹具。

### 🚨 致命缺陷与解决方案
如果直接用这块板子，你**必须**解决以下缺失功能（核心板通常不带）：

| 缺失模块 | 为什么会有问题 | 解决方案 (必须做一块“底板”) |
| :--- | :--- | :--- |
| **1. 听觉与发声** | 开发板没有麦克风和功放，无法直接说话。 | 绘制一块**“音频底板”**，板载 2颗麦克风、MAX98357A 功放和喇叭接口。 |
| **2. 电池充电** | 该开发板通常只支持 USB 5V 供电，**不带锂电池充电管理**。 | 在“底板”上集成 **TP4056** 或类似充电芯片，否则公仔只能插线玩。 |
| **3. 结构干涉** | 开发板是长条形且 USB 接口位置固定。 | 你的公仔必须根据开发板的 USB 位置开孔，限制了造型自由度。 |
| **4. 震动脱落** | 只是插针连接，运输或摔打容易松动。 | 生产时必须**焊接**排针，不能只用插座；或者打热熔胶固定。 |

### 📉 成本账 (每台)
- **全定制 PCBA 方案**：ESP32模组(¥20) + 音频/电源IC(¥10) + PCB(¥5) = **约 ¥35-40 元**
- **开发板量产方案**：开发板(¥46) + 音频/电源底板(¥15) + 连接件 = **约 ¥65-70 元**

**结论**：
**可行**。每台成本多花约 25 元，500 台总成本多花 ¥12,500。
**建议**：如果你极度缺乏硬件设计信心，愿意花这 1.25 万元来买个“安心”，那就采用此方案。**但你依然需要设计一块“音频电源扩展底板”来插这块开发板。**
