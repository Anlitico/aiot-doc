## 2. 前期固定投入 (One-time Investment)

在正式出货前，需要支付的非经常性工程费 (NRE)：

1.  **结构设计服务费**：约 ¥3,000 - ¥5,000 (外包工程师)。
2.  **硅胶模具费 (铝模)**：约 ¥8,000 - ¥12,000 (支持 5k-1w 次翻模)。
3.  **PCB 网板/治具费**：约 ¥600。
4.  **功能打样费 (3D打印+板拔)**：约 ¥1,500。

**总投入：约 ¥13,000 - ¥19,000**

---

## 3. 500 台首批启动总资金 (Cash Flow)

若 500 台一次性投产：
- **物料采购**：¥208 * 500 = ¥104,000
- **固定投入**：¥16,000
- **不可预见支出 (5%)**：¥6,000

**启动总预算建议：¥126,000 (约 13万人民币)**

---

## 4. 盈利模型分析

- **方案 A (极客版)**：定价 ¥499。毛利率约 58%，回本点约 65 台。
- **方案 B (艺术潮玩版)**：定价 ¥899。毛利率约 77%，回本点约 25 台。
- **结论**：由于 IP 属性强（名人性格），建议高定价，以覆盖后续云端 API (OpenAI/ElevenLabs) 的长期调用成本。
