# 3D 建模与结构设计任务书 (3D Modeling & MD Design Brief)

**项目名称**：AI 语音公仔 (基于 ESP32-S3 平台)
**设计语言**：双瓣异形、硬质硅胶蒙皮、内部塑料骨架。

---

## 1. 核心设计输入 (Design Constraints)

### 1.1 电子零件外形
- **主 PCB**：垂直放置，35mm (宽) x 55mm (高)，厚度 1.2mm。需预留背部 USB-C 接口开孔对齐。
- **副 PCB (x2)**：圆形直径 22mm。需放置在公仔两侧圆环正中心。
- **电池**：1000mAh 软包锂电池，建议尺寸 103048 (10mm高 x 30mm宽 x 48mm长)。
- **喇叭**：一体化音腔模组，建议尺寸 25mm x 35mm x 8mm。

### 1.2 材质约束
- **蒙皮**：硬质硅胶 (Shore A 70)。壁厚要求：主体 2.2mm，圆环导光区减薄至 0.8mm。
- **骨架**：PC+ABS 材质。要求采用“天线净空设计”，顶部 5mm 内禁止出现金属件。

---

## 2. 关键结构难点 (Hard Requirements)

### 2.1 隐形隐藏式按键 (Hidden Button)
- 背部需设计 POM 材质传力柱，通过按压硅胶壳触发 PCB 上的轻触开关。
- 要求：按压行程 0.3mm-0.5mm，需有明显的力学反馈。

### 2.2 声学密封与减震
- **麦克风舱**：必须设计专用的密封胶套压紧位。
- **音腔悬浮**：喇叭卡槽需预留 0.5mm 泡棉压缩余量。
- **气压平衡**：底座需设计一个隐藏的 0.5mm 呼吸孔。

### 2.3 装配逻辑 (DFA)
- 必须按“内胆总装 -> 套入蒙皮”的逻辑设计。
- 所有零件采用卡扣或少量 M2 螺丝固定。

---

## 3. 交付物要求 (Deliverables)

1. **3D 模型文件**：完整的结构总装图 (`.stp` 或 `.igs` 格式)。
2. **零件 2D 图纸**：包含关键尺寸标签、公差要求及材质表面说明。
3. **炸开图说明 (Exploded View)**：用于指导生产线装配。
4. **BOM 结构表**：所有的结构件（螺丝、泡棉、塑胶件）清单及参考克重。

---

## 4. 工程师验收准则 (Acceptance Criteria)

- [ ] **干涉检查**：所有零件装配后无物理干涉。
- [ ] **接口对齐**：USB-C 接口中心线与外壳开口误差 < ±0.1mm。
- [ ] **壁厚检查**：确保没有由于壁厚不均导致的注塑缩水风险。
- [ ] **重量平衡**：质心需处于底部 1/3 位置，确保垂直重力自稳。
