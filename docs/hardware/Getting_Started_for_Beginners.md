# 硬件开发小白快速入门指南 (Beginner's Hardware Roadmap)

如果你没有硬件背景，但想亲自开始本项目的第一版 PCB 设计，请遵循以下流程。

---

## 1. 工具准备：嘉立创 EDA (EasyEDA) 专业版

不要学复杂的专业软件（如 Altium 或 Cadence）。直接使用 [嘉立创 EDA 专业版](https://pro.lceda.cn/)。
- **一键开路**：它将“设计”、“买零件 (LCSC)”、“打板 (JLC)”全部打通。
- **元件库**：搜索 `C467406` (ESP32-S3 模组)，它会自动带出准确的封装图，你不用自己画。

---

## 2. 硬件设计的三个阶段

### 第一阶段：原理图 (Schematic) - 逻辑连线
就像画流程图。定义哪个引脚连到哪个零件。
- **高阶技巧**：搜索 **“ESP32-S3-DevKitC-1 参考原理图”**。
- **必做工作**：按照我们的 `Pin_Mapping_Guide.md` 将引脚与 I2S 功放、麦克风 FPC 连起来。

### 第二阶段：PCB 布局 (Layout) - 物理落位
将原理图的逻辑变成实际的铜线。
- **形状控制**：设置板框尺寸为 **35mm x 55mm**。
- **分层建议**：初学者建议直接做 **4 层板**。虽然贵一点点，但信号极其稳定，怎么画都很难出错。

### 第三阶段：一键下单 (PCBA)
设计完成后，点击“导出制板文件 (Gerber)”，然后在嘉立创下单。
- **核心建议**：选择 **PCBA 先进产线**（哪怕只买 5 片）。他们会帮你把那些细小的芯片全部焊好，你拿到的直接就是能开机的成板。

---

## 3. 给新手的 3 条忠告

1.  **天线不要有金属遮挡**：在 PCB 布局时，ESP32 模组的天线部分必须要伸出板子外面，或者天线下方所有层都是“开窗”不铺铜的。
2.  **电源线要画得粗**：所有 3.3V 和 5V 的线，宽度建议设为 **15-20mil**，其余信号线可以用 **6-8mil**。
3.  **电容要靠近引脚**：你在原理图里看到的那些 0.1uF 小电容，在 PCB 上必须“死死贴着”芯片的电源引脚。

---

## 4. 下一步动作建议

**如果你现在想尝试：**
1. 注册一个“嘉立创 EDA”账号。
2. 在开源广场搜索 `Korvo-1` 或者 `ESP32-S3 AI voice`。
3. 观察他们的走线，尝试把它们“缩小”到我们的支架尺寸内。
