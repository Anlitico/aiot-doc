# ESP32-S3 天线选型与外观适配全指南

在智能语音玩具设计中，Wi-Fi 信号的稳定性直接决定了对话的实时感。本指南详细对比了 ESP32-S3 模组的两类天线方案。

## 1. 方案对比：PCB 天线 vs. IPEX 外部天线

| 特性 | 板载 PCB 天线 (WROOM-1) | IPEX 外部天线 (WROOM-1U) |
| :--- | :--- | :--- |
| **形态** | 模组自带蛇形铜箔天线 | 模组带微型座子，需外接天线 |
| **BOM 成本** | $0 (包含在模组内) | 需额外采购天线 + 连接线 (约 ￥1-2) |
| **组装难度** | 无需组装，贴片即完成 | 需人工扣合 IPEX 连接头，耗时 |
| **可靠性** | 极高 (不会松脱) | 中等 (剧烈震动或跌落可能脱落) |
| **信号性能** | 良好 (取决于 PCB 布局) | 优秀 (可避开障碍物，增益更高) |
| **适用场景** | 空间充足、外壳非金属、成本敏感 | PCB 被深埋、空间极其局促、信号要求极高 |

---

## 2. 深度选型建议：如何根据“海绵宝宝”适配

### 2.1 什么时候选 PCB 天线？(首选推荐)
如果你的“海绵宝宝”内部结构允许 PCB 靠近身体的某一侧边缘。
- **空间要求**：天线周边的 PCB 区域不能铺铜（Keep-out 区域）。
- **材质要求**：外壳为普通塑料、硅胶、织物。
- **布局要点**：模组的天线部分应尽可能靠近玩具的“皮肤”。

### 2.2 什么时候选 IPEX 外部天线？
- **硬件堆叠拥挤**：如果海绵宝宝内部塞满了喇叭、大容量电池、金属骨架，且 PCB 被挤在正中心。
- **异形结构限制**：如果 PCB 只能水平放置在底部，而信号需要向上穿透厚重的填充物。
- **操作建议**：采购 **2.4GHz FPC 软膜天线**。其自带背胶，可以像贴纸一样贴在玩具头顶或背后的内侧。

## 3. 工业设计 (ID) 与声学设计的避坑指南

### 3.1 净空区 (Keep-out Area)
无论是哪种天线，其辐射的正前方 **15mm 范围内** 严禁出现以下物品：
1.  **电池**（巨大的金属屏蔽体）。
2.  **喇叭/扬声器**（含有磁铁和金属线圈）。
3.  **连接线束**（杂乱的电线会变成干扰源）。
4.  **人手接触区**（人体是巨大的吸波体，如果手抓的地方正好是天线，信号会掉一格）。

### 3.2 材质黑名单
制作海绵宝宝外壳时，请避免：
- **金属漆/真空电镀**：任何含金属粉末的涂装都会形成“法拉第笼”，阻断 Wi-Fi。
- **导电橡胶**：部分黑色橡胶件可能导电。
- **过厚的外壳**：外壳厚度建议控制在 3.0mm 以内。

---

## 4. 总结：给你的行动建议
1.  **初期原型**：继续使用 **PCB 天线** 版本的开发板（Korvo-1 就是 PCB 天线的）。
2.  **ID 定稿阶段**：测量 PCB 距离外壳的距离。
    - 如果 > 10mm 且中间有障碍物：改为 **WROOM-1U (IPEX)**。
    - 如果 < 5mm 且周围空旷：坚持 **WROOM-1 (板载)**。
3.  **测试动作**：在海绵宝宝内部装好所有零件，用手机热点测试接收信号强度 (RSSI)，若 RSSI 低于 -70dBm，说明必须更换外部天线。
