# PCB 物理整合与系统架构 (PCB Physical Integration)

本指南为“双瓣”造型玩具的电路系统提供物理实现标准。针对初创项目，我们采用**分布式母女板架构**以平衡结构嵌套与信号质量。

---

## 1. 系统拓扑：一主两副 (Distributed Architecture)

### 1.1 中央主控板 (Main Logic Board)
- **定位**：垂直高度中心。
- **任务**：ESP32-S3 核心演算、Wi-Fi 天线、3.3V/800mA 供电、2W 音频功放、背部 USB 接口。
- **物理规格**：35mm x 55mm 长条形，底部通过 PH2.0 插座连接电池和喇叭。

### 1.2 传感器圆环副板 (Sensor Daughter-boards x 2)
- **定位**：左右圆环中心。
- **任务**：12 颗 WS2812B RGB 灯环（正面）、1 路数字麦克风（背面）。
- **物理规格**：圆形，单面贴装。由于移除了物理按键，厚度可进一步压缩至 1.2mm。

---

## 2. 互连协议：FPC 柔性连接 (Interconnects)

- **线序定义 (6-Pin)**：
  1. VCC (3.3V)
  2. GND
  3. WS2812_DATA
  4. MIC_BCK
  5. MIC_WS
  6. MIC_DATA
  *(保留 6-pin 以适应标准连接器，若需缩减，可使用 4/5-pin)*

---

## 3. 防护与热管理 (Protection & Thermal)

### 3.1 玩具级 ESD (静电防护)
硅胶外壳摩擦产生的静电极高。
- **强制要求**：副板的 FPC 入口处必须增加 **TVS 保护阵列**（如 SRV05-4）。
- **屏蔽**：在副板背面（对应麦克风位置）铺地层，作为静电泄放参考面。

### 3.2 热物理隔离
- **大热源 (S3 模组)** 放置在主控板顶部，利用支架上方的空气室对流散热。
- **敏感源 (锂电池)** 放置在底座独立舱体内，中间通过塑料支架壁物理隔断，确保电池温升不超过 5℃ 以上。

---

## 4. 生产装配坐标 (Assembly Coordinates)

为了确保背部 USB 接口不歪斜，PCB 设计时需遵循：
- **坐标原点 (Offset)**：以 USB-C 接口中心线为物理原点。
- **限位结构**：主板左右两侧设计“定位耳”，卡入内部支架的限位槽，误差需控制在 ±0.1mm 内。

---

## 5. 成本与方案权衡 (Cost & Tradeoff Analysis)

基于深圳电子产业链 500 套试产规模，对**分布式母女板**方案进行成本核实与选型记录。

### 5.1 成本拆解 (每台增加成本预估)

| 项目 | 预估单价 (RMB) | 包含内容 |
| :--- | :--- | :--- |
| **硬件 BOM 增量** | ¥5.0 - ¥8.0 | 2块圆形 PCB 副板、FPC 连接座、TVS 保护管。 |
| **FPC / 连接线束** | ¥2.5 - ¥4.0 | 2条定制长 FPC 软排线（含开模摊销）。 |
| **加工与良率损失** | ¥2.5 - ¥3.0 | 额外的贴片工位费及 FPC 组装人工。 |
| **合计增量** | **¥10 - ¥15 / 台** | 相较于单块大硬板的总增量。 |

### 5.2 为什么坚持使用“母女板”方案？

即使成本增加约 12 元，仍推荐以此作为试产标准，理由如下：
1.  **物理过弯装配**：单块大 PCB（十字型）无法从窄小的颈部接口塞入。母女板可分别塞入并就位。
2.  **麦克风近场拾音**：副板让麦克风直接贴合圆环导音孔。若使用单板，声音需通过长达 30mm 的导管，会导致 AEC 算法失效。
3.  **触摸灵敏度保证**：副板紧贴圆环内侧，触摸感应电极行程最短，可有效抑制干扰。

### 5.3 备选降本方案 (Budget Alternative)
若后续总成本超支，可采取以下降本动作：
- **方案**：保留“母女板”逻辑，但撤掉 **FPC 软排线**，改用普通的 **PH1.0 电子线束+插头**。
- **代价**：线束较厚且硬，可能导致硅胶壳表面在“大腿/躯干”结合处产生不规则凸起。
