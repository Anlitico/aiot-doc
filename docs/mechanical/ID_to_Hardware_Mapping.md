# 外观交互与硬件定义映射表 (ID to Hardware Mapping)

本文档旨在协调海绵宝宝的“长相”与内部“引脚/布局”的对应关系。

---

## 1. 感官落位映射 (Sensory Layout)

| 外观部件 (ID Part) | 对应功能 | 硬件载体 | PCB/引脚 约束 |
| :--- | :--- | :--- | :--- |
| **侧面圆环 (左右对称)** | 语音拾取 | 2 x MEMS 麦克风 | 位于圆环中心，需采用 FPC 或副板连接。 |
| **侧面圆环 (边缘)** | 视觉反馈 | 12 颗 RGB LED 环 | 围绕圆环布置灯带，引出 3-pin 控制线。 |
| **侧面圆环 (表面)** | 捏一捏唤醒 | 电容式触摸感应 | 敷设金属箔或弹簧，连接至 S3 Touch 引脚。 |
| **背部下格栅** | 语音输出 | 4Ω 3W 喇叭 + 音腔 | 音腔出音孔需对准格栅。 |
| **背部中心接口** | 充电与控制 | USB-C + 物理按键 | 垂直主板后端直接出头。 |

---

## 2. 材质对引脚与布局的影响

### 2.1 长毛绒/厚硅胶材质
- **影响**：散热极差。
- **对策**：PCB 布局需将 ESP32-S3 模组（热源）与音频功放（热源）物理隔离。
- **引脚提示**：若检测到外壳内部热积聚严重，可能需要预留一个 NTC（热敏电阻）引脚进行温度监控。

### 2.2 内部填充物 (棉花/海绵)
- **影响**：吸声与压力。
- **对策**：麦克风必须加装**密封套**。如果直接埋在棉花里，声音会发闷且 AEC 算法失效。
- **布局提示**：PCB 两侧麦克风区域需预留硅胶防震垫的粘连位。

---

## 3. 物理冲突确认 (Collision Check)

1.  **天线净空区 vs 材质装饰**：若海绵宝宝背部有金属质感的涂装，PCB 天线必须调头朝向前胸。
2.  **电池厚度 vs PCB 组件高度**：1000mAh 电池厚度约为 6mm。若 PCB 底部也有高组件（如大电解电容），电池将无法贴合，会导致公仔“挺大肚子”。

---

## 4. 下一步行动建议

1. **定原样**：确定海绵宝宝的身高和腰围（15cm x 15cm 是常用黄金比例）。
2. **定接口**：硬件工程师根据 ID 提供的“开孔位置”调整 PCB 的 USB 和按键落位。
3. **定连接**：确定哪些外设是“板载”的，哪些是“线连”的（线连的要多做排针/插座）。
